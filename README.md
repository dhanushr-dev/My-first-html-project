# My Multimedia Blog 🎬🎵

Welcome to **My Multimedia Blog**, sponsored by **TAP Academy™**. This is a responsive, single-page HTML project showcasing a collection of favorite audio tracks and a curated list of the top 3 most-watched Indian film trailers on YouTube.

---

## 🚀 Live Demo
🌐 **GitHub Pages**: [dhanushr-dev.github.io/My-first-html-project](https://dhanushr-dev.github.io/My-first-html-project/)

---

## 🚀 Features

- **Favorite Songs Showcase**: Play and control favorite audio tracks directly on the webpage using integrated HTML5 audio controls.
- **Top 3 Most-Watched Indian Film Trailers**: 
  - **Salaar Part 1: Ceasefire** (133.2 Million Views, ₹600+ Crore Box Office)
  - **KGF Chapter 2** (124.7 Million Views, ₹1200+ Crore Box Office)
  - **Animal** (114.7 Million Views, ₹900+ Crore Box Office)
- **High-Quality Media**: Features movie poster images, custom video trailer players with preview thumbnails/posters, and detailed synopses for each film.
- **Semantic HTML5 Elements**: Built using semantic elements like `<audio>`, `<video>`, `<mark>`, `<abbr>`, `<del>`, `<strong>`, and `<em>` for proper structure and accessibility.
